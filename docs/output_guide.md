# Outputs and Screenshots Guide 📸

To document this project effectively, you should capture and save the following items in an `images/` folder:

### 1. Project Structure
Capture a screenshot of your VS Code file explorer to show the organized folder structure (`data/`, `src/`, `docs/`, etc.).

### 2. Dataset Preview
Open the generated `data/mock_expenses.csv` and take a screenshot of the raw data.

### 3. Terminal Output
Run `python main.py` and capture the log showing database initialization and data seeding.

### 4. Dashboard Visuals
In the Streamlit dashboard, capture:
- **KPI Metrics**: The Total Spend/Income/Savings cards.
- **Monthly Trend**: The line chart showing spending over time.
- **Category Split**: The bar chart showing where the most money is spent.

### 5. Final Report
Open a generated Excel file from the `reports/` folder and screenshot the "Category Summary" sheet.

### 6. GitHub Repository
Once uploaded, take a screenshot of your GitHub repo homepage showing the clean README and documentation.
